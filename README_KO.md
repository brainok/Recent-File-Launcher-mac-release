# Recent File Launcher

Recent File Launcher는 macOS 메뉴 막대에서 최근 사용한 파일, 다운로드한 파일, 저장한 파일을 빠르게 찾고 여는 앱입니다. Finder나 여러 폴더를 직접 뒤지지 않고 최근 파일 흐름을 한곳에서 확인해 반복적인 파일 열기 작업을 줄여 줍니다.

## 다운로드

최신 DMG:

https://github.com/brainok/Recent-File-Launcher-mac-release/releases/latest/download/Recent.File.Launcher.dmg

## 설치

1. `Recent.File.Launcher.dmg`를 엽니다.
2. `Recent File Launcher.app`을 `Applications` 폴더로 드래그합니다.
3. `/Applications/Recent File Launcher.app`을 실행합니다.

macOS가 첫 실행 보안 경고를 표시하면 Finder에서 앱을 우클릭한 뒤 `열기`를 선택하고, 다시 한 번 `열기`를 선택합니다.

## 주요 기능

- 최근 다운로드 또는 저장 파일 빠르게 열기
- 메뉴 막대에서 필요한 파일에 바로 접근
- 파일 검색과 최근 파일 목록 확인
- 반복적인 Finder 탐색 시간 줄이기
