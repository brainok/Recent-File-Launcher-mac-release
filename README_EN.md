# Recent File Launcher

Recent File Launcher is a macOS menu bar app for quickly finding and opening recently used, downloaded, or saved files. It helps reduce repeated Finder navigation by keeping recent file access close at hand.

## Download

Latest DMG:

https://github.com/brainok/Recent-File-Launcher-mac-release/releases/latest/download/Recent.File.Launcher.dmg

## Install

1. Open `Recent.File.Launcher.dmg`.
2. Drag `Recent File Launcher.app` into `Applications`.
3. Launch `/Applications/Recent File Launcher.app`.

If macOS asks for file access or privacy permissions, allow them in System Settings.

## Key Features

- Quickly open recently downloaded or saved files
- Access recent files from the macOS menu bar
- Search and review recent file lists
- Reduce repeated Finder navigation
