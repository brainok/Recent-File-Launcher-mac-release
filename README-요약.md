# README-요약

Recent File Launcher는 macOS 메뉴 막대에서 최근 사용, 다운로드, 저장한 파일을 빠르게 찾아 여는 앱입니다. Finder에서 여러 폴더를 반복해서 찾는 시간을 줄이고, 자주 이어지는 파일 작업을 더 가볍고 빠르게 처리할 수 있도록 돕습니다.

Recent File Launcher is a macOS menu bar app that helps you quickly find and open recently used, downloaded, or saved files. It reduces repeated Finder navigation and keeps everyday file access fast and close at hand.
